## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the app locally:**
   ```bash
   node server.js
   ```
   Then open [http://localhost:3000](http://localhost:3000) in your browser.

3. **Build & Deploy:**
   - The app is ready to be deployed as a static site or Node.js server.
